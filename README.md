
# **Project Name:** Learn Turkish

## **Description**

Learn Turkish is an interactive platform designed to help users learn the Turkish language effectively. The website features engaging lessons, vocabulary-building tools, and interactive content, making language learning fun and accessible.

## **Key Features:**

1.  **Welcome Page:** Greets users with their name if logged in.
2.  **Navbar:** Easy navigation with links like _Home_, _Start Learning_, _Tutorials_, _About Us_, and _My Profile_.
3.  **Language Lessons:** Lessons displayed on visually appealing cards.
4.  **Success Tracker:** Tracks progress with animated counters (e.g., lessons completed).
5.  **Responsive Design:** Optimized for mobile and desktop users.

## **Technologies Used:**

-   **React Router:** For smooth navigation between pages.
-   **Firebase Authentication:** For user login and signup.
-   **React Context API:** For managing user data and app state.
-   **Tailwind CSS:** For creating a modern and responsive design.
-   **React Animations:** For success tracking and enhancing user experience.
